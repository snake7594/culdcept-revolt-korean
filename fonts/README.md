# 폰트 폴더

릴리즈 패치의 한글 글리프는 **나눔스퀘어 네오 Bold**(NanumSquare Neo, © NAVER Corp.)로
렌더링합니다. 이 폰트 파일은 저작권상 저장소에 포함하지 않습니다.

파이썬 툴(`apply_korean_opening.py`)로 **릴리즈와 동일한 모양**을 만들려면, 나눔스퀘어 네오
Bold 파일을 이 폴더에 아래 이름으로 두세요:

```
fonts/NanumSquareNeo-cBd.ttf
```

- 나눔스퀘어 네오는 네이버에서 무료 배포합니다: <https://hangeul.naver.com/font>
- 파일이 없으면 툴은 시스템 한글 폰트(예: 윈도우 맑은 고딕)로 대체합니다.
  `--font <경로>` 로 원하는 TTF를 직접 지정할 수도 있습니다.

폰트 라이선스는 배포처(네이버)의 약관을 따르세요. 이 저장소는 폰트 파일을
재배포하지 않으며, 게임 내 비트맵 폰트에 글리프를 임베드(래스터화)해 쓸 뿐입니다.
